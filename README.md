# 💄 BeautyStore - E-Commerce Homepage

This is a simple, conversion-optimized homepage for a **beauty products e-commerce website**, built using **React** and **Bootstrap 5**.

---

## 🛍️ Project Features

- 🔥 **Hero Product Section** for the best-selling "All-in-One Hairoil"
- 📦 Product categories displayed using **responsive Bootstrap cards**
- ✅ “Why Choose Us” section to increase trust (CRO-focused)
- ✉️ Newsletter signup for lead generation
- 🖼️ Placeholder images for all 8 beauty products
- 📱 Fully responsive (mobile, tablet, desktop)

---

## 🧪 Tech Stack

- **React**
- **Bootstrap 5**
- **HTML5/CSS3**
